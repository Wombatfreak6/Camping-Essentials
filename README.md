# Camping Essentials Guide

Camping Essentials Guide is a simple and responsive web page that highlights important camping and survival topics such as shelters, firecraft, food procurement, and more. It is built using only HTML and CSS with a clean, natural green design theme.

## Features
- Responsive layout using Flexbox
- Lightweight design with no external CSS frameworks
- Custom favicon and consistent branding
- Natural green color palette for an outdoor feel

## Technologies Used
- HTML5  
- CSS3 (Flexbox)  
- Google Fonts (Nunito)

## How to Use
1. Download or clone the project folder.
2. Open the `index.html` file in your browser.
3. Customize the text, colors, or icons as needed.

## Files
- `index.html` – main webpage file  
- `style.css` – custom styling file  
- `README.md` – project documentation
